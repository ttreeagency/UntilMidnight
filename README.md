# UntilMidnight

A small Fusion helper to get the number of seconds before midnight

Usage
-----

    prototype(Your.Package:Content) {
        @cache {
            mode = 'cached'
            maximumLifetime = Ttree.UntilMidnight:Seconds
        }
    }
